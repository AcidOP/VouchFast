# Introduction

## 1. Get All Testimonials

Retrieve all testimonials from all lists associated with the authenticated user.

```http
GET https://vouchfast.vectorized.tech/api/testimonials
```

### Request Headers

| HEADER  | TYPE   | REQUIRED | DESCRIPTION                                                |
| ------- | ------ | -------- | ---------------------------------------------------------- |
| Api-Key | string | true     | A valid [API key](/dashboard/api-keys) for authentication. |

### Request

```javascript
const url = 'https://vouchfast.vectorized.tech/api/testimonials';

fetch(url, {
  method: 'GET',
  headers: {
    'Api-Key': 'vf_abcdefghijklmnopqrstuvwxyz1234',
  },
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
```

## 2. Get a specific Testimonial

```http
GET https://vouchfast.vectorized.tech/api/lists/:id
```

### Request Headers

| HEADER  | TYPE   | REQUIRED | DESCRIPTION                                                |
| ------- | ------ | -------- | ---------------------------------------------------------- |
| Api-Key | string | true     | A valid [API key](/dashboard/api-keys) for authentication. |

### Request

```javascript
const url =
  'https://vouchfast.vectorized.tech/api/testimonials/67457810bce6ffb04a3aafba';

fetch(url, {
  method: 'GET',
  headers: {
    'Api-Key': 'vf_abcdefghijklmnopqrstuvwxyz1234',
  },
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
```

### Success Response

```json
{
  "success": true,
  "data": {
    "id": "67790ed73beae625b0031805",
    "name": "Arnold Schwarzenneger",
    "message": "Good job with the bulking!",
    "email": "arnie@mail.com",
    "listId": "6724f7df10d485ef260ea9fb",
    "createdAt": "2025-01-04T10:35:03.231Z",
    "updatedAt": "2025-01-04T10:35:48.566Z"
  }
}
```

### Error Responses

| Status Code | Meaning               | Example                                                               |
| ----------- | --------------------- | --------------------------------------------------------------------- |
| 401         | API key is required.  | `{ success: false, error: 'API Key is required' }`                    |
| 404         | No lists found.       | `{ success: false, error: 'Lists not found' }`                        |
| 500         | Internal server error | `{ success: false, error: 'Failed to fetch testimonials from list' }` |
