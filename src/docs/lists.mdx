# Introduction

Lists are collections of testimonials. This API allows you to retrieve all lists or fetch testimonials from a specific list.

## 1. Get All Lists

Retrieve all testimonial lists associated with the authenticated user.

```http
GET https://vouchfast.vectorized.tech/api/lists
```

### Request Headers

| HEADER  | TYPE   | REQUIRED | DESCRIPTION                                                |
| ------- | ------ | -------- | ---------------------------------------------------------- |
| Api-Key | string | true     | A valid [API key](/dashboard/api-keys) for authentication. |

### Request

```javascript
const url = 'https://vouchfast.vectorized.tech/api/lists';

fetch(url, {
  method: 'GET',
  headers: {
    'Api-Key': 'vf_abcdefghijklmnopqrstuvwxyz1234',
  },
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
```

### Succcess Response

## 2. Get Testimonials from a Specific List

Retrieve all testimonials within a given list.

```http
GET https://vouchfast.vectorized.tech/api/lists/:id
```

### Request Headers

| HEADER  | TYPE   | REQUIRED | DESCRIPTION                                                |
| ------- | ------ | -------- | ---------------------------------------------------------- |
| Api-Key | string | true     | A valid [API key](/dashboard/api-keys) for authentication. |

### Request

```javascript
const url = 'https://vouchfast.vectorized.tech/api/lists/674c092d7zf97c90e2cdb7d4';

fetch(url, {
  method: 'GET',
  headers: {
    'Api-Key': 'vf_abcdefghijklmnopqrstuvwxyz1234',
  },
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
```

### Error Responses

| Status Code | Meaning               | Example                                                               |
| ----------- | --------------------- | --------------------------------------------------------------------- |
| 401         | API key is required.  | `{ success: false, error: 'API Key is required' }`                    |
| 404         | No lists found.       | `{ success: false, error: 'Lists not found' }`                        |
| 500         | Internal server error | `{ success: false, error: 'Failed to fetch testimonials from list' }` |
